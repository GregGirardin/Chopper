A 2D sprite helicopter game.

Requires PIL. Currently in Python 2.

Objective: Complete all levels by destroying all enemies and then returning to base.
           Defend the city. If the city is destroyed the game ends. The city can be bombed by planes
           or attacked by vehicles.
           Landing on the base will replenish weapons and health.

Keys:

up/down/left/right to manuver chopper
a: small missile
s: large missile
z: bomb
sp: bullet
e/d: aim cannon
?: instructions/pause

Run main.py
